🧩 Core Idea (Hybrid Document + Flow render)

Build an editor where users can work in two connected modes:

📝 Normal document mode

Write text like Notion.

Structured: headings, paragraphs, lists, and existing structure.

🎨 Flow mode

The same content appears as movable blocks.

Users can draw, connect, group, and rearrange ideas visually like Figma.

Add diagrams, arrows, sketches, and “subspaces” (other-documents).

🔄 Smart Bridge Between Them

Visual changes -> automatically reorganize the document.

Drawings/diagrams -> become structure, text, or embedded visuals.

Users can choose how visuals convert back to text.

Optional complexity (simple for normal users, powerful for advanced users).
