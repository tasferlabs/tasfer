<p className="dx-lede">`@tasfer/editor` is a CRDT-first canvas editor for the web — the same engine that powers the Tasfer app, published as a standalone, framework-agnostic package. **It renders to a canvas, models your document as a conflict-free replicated data type, and never assumes a server exists.**</p>

<h2 id="install">Install</h2>

Add the package with your package manager of choice:

<InstallTabs pkg="@tasfer/editor" />

The core package has **zero runtime dependencies**. The CRDT, the canvas renderer, and the markdown serializer all ship inside it. You only add more packages when you want a framework binding or a network provider:

<InstallTabs pkg="@tasfer/react @tasfer/provider-relay" />

<h2 id="requirements">Requirements</h2>

- **A browser with `ResizeObserver` and a 2D `<canvas>` context** — every evergreen browser since ~2020. There is no DOM-contenteditable fallback by design; the canvas *is* the editor.
- **ES2020 or a bundler that targets it.** The package ships ESM with a CommonJS interop entry. Types are bundled — no `@types` install.
- **No build step required for the CRDT.** It's pure JavaScript — there's nothing to copy into your `public/` folder.

<h2 id="first">A thirty-second editor</h2>

The smallest thing that works — an editable document with markdown shortcuts, history, and a blinking cursor. In plain JS you mount the engine into an element you own; in React you render the `<Editor>` component, which owns and sizes its host for you and reports edits through `onChange`. Flip the tab:

<FrameworkTabs>

```ts file="main.ts"
import { createEditor } from "@tasfer/editor";

const editor = createEditor({
  element: document.querySelector("#editor")!,
  markdown: "# Hello\n\nStart typing — *markdown* shortcuts just work.",
  autofocus: true,
});

// Read it back at any time, as plain markdown:
editor.on("change", () => {
  localStorage.setItem("draft", editor.getMarkdown());
});
```

```tsx file="App.tsx"
import { Editor } from "@tasfer/react";

export default function App() {
  return (
    <Editor
      markdown={"# Hello\n\nStart typing — *markdown* shortcuts just work."}
      autofocus
      style={{ height: "70vh" }}
      // Read it back as plain markdown on every edit:
      onChange={(markdown) => localStorage.setItem("draft", markdown)}
    />
  );
}
```

</FrameworkTabs>

<Callout kind="tip" title="That document is already a CRDT.">Even with no provider attached, the editor's value is backed by a replicated data structure. Wiring up sync later is additive — you never rewrite your document model to "make it collaborative."</Callout>
