To build from source:

```bash
git clone https://github.com/hamza512b/cypher
cd cypher
pnpm install
pnpm dev          # opens the app at http://localhost:4000
```

<h2 id="first-doc">Your first document</h2>

<Steps>
  <Step title="Pick a folder">
    On first launch, Cypher asks for a folder. It mirrors every page into it as plain `.md` files — a **one-way** copy you can open in any editor, today or in twenty years. The app keeps the source of truth; the folder is always yours.
  </Step>
  <Step title="Write">
    Markdown shortcuts work as you type: `#` for headings, `**bold**`, `>` for quotes, `- [ ]` for tasks. Everything is saved locally the instant you type — there is no "save" button and no spinner reporting to a server.
  </Step>
  <Step title="Back it up like any folder">
    Because the mirror is just files, your existing backup tools already cover it. Copy it, zip it, put it in a git repo, sync it with rsync. The mirror is one-way — Cypher writes to the folder, it never reads your edits back from it.
  </Step>
</Steps>

<Callout kind="note" title="No telemetry, ever.">Cypher ships with no analytics, no crash reporting that phones home, and no account system. The first run does not contact a network at all. See [Privacy & data](/docs/app/privacy) for the complete list of what the app does and does not collect.</Callout>
