<h2 id="reading">قراءة الحالة</h2>

لا يسلّمك المحرر مخزنًا قابلًا للتغيير. `state` لقطة جديدة غير قابلة للتغيير تُعاد بناؤها عند كل قراءة، وتسقط `getMarkdown()` المستند الحالي إلى نص. يمثل **وجه `query`** مرآة القراءة لـ[`change`](/docs/editor/api-commands#targets): تتحدث كل دالة **لغة `DocPoint` و`DocRange` نفسها التي تقبلها [ChangeApi](/docs/editor/api-commands#targets)**، ويكون افتراضها المؤشر أو التحديد الحي. تعكس `block` و`blocks` فصل النقطة عن النطاق في `change`؛ فتقترن `query.block(at)` بـ`setBlock(attrs, at)`، وتقرأ `query.blocks(range)` النطاق الذي يغطيه التحديد. وهكذا تقرأ موضعًا ثم تمرره مباشرة إلى تعديل من دون الوصول إلى الحالة الخام. ويقيم التحديد نفسه (أي `DocRange` للمؤشر أو المرساة مع الإزاحات) في اللقطة التفاعلية عند `state.selection.range`.

<h2 id="writing">تغيير المستند</h2>

```ts

  c.setBlock({ type: "heading", level: 1 }).insertText("Untitled"),
);
```

]} />

```ts
import { OPEN_LINK } from "@tasfer/editor";

// Override a built-in action: open links through the host's own router
// instead of the engine's default. Returning true claims it (stops the
// default); return void to observe and let the default still run.
const off = editor.registerAction(OPEN_LINK, ({ url }) => {
  router.push(url);
  return true;
});

// later — detach the handler
off();
```

<h2 id="events">الأحداث</h2>

تشترك `on(event, cb)` وتعيد دالة إلغاء اشتراك. يتلقى مستمع `change` كائن `ChangeTransaction` ‏(`{ isRemote, ops }`)؛ ويتلقى الآخرون `EditorStateSnapshot`، وهي القيمة غير القابلة للتغيير نفسها في `editor.state`.

<PropsTable cols={["الحدث", "وقت الإطلاق", "الحمولة"]} rows={[
  { name: "change", type: "ChangeTransaction", desc: "عند تغير المستند بالكتابة أو اللصق أو التراجع أو المزامنة البعيدة. الحمولة { isRemote, ops }." },
  { name: "selectionchange", type: "EditorStateSnapshot", desc: "عند تحرك المؤشر أو التحديد من دون تعديل المستند." },
  { name: "focus / blur", type: "EditorStateSnapshot", desc: "عند اكتساب لوحة الرسم التركيز أو فقده." },
]} />

```ts
const off = editor.on("change", (tx) => {
  if (tx.isRemote) return;            // ignore edits that came from a peer
  localStorage.setItem("draft", editor.getMarkdown());
});
// later
off();
```

<Callout kind="note" title="لا يوجد حدث `sync` على المحرر.">تأتي حالة الاتصال وعدد الأقران من *المزود* لا المحرر؛ فالمزود يملك النقل. اشترك باستخدام `provider.on("sync", …)`. راجع [التعاون](/docs/editor/collaboration).</Callout>

const editor = createEditor({ element });
